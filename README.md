# webchat-view

## node安装

### linux安装

1、首先你需要到 node 官网 去下载一个 linux 可用的 node 版本。

```
https://nodejs.cn/download/
```

2、移动 tar 文件

```
mv /home/www/lightning-client/node-v18.18.0-linux-x64.tar.xz ./
```

npm install
